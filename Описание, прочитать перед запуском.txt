Информация перед запуском: большинство time.sleep оставлены для визуальной демонстрации, некоторые необходимы из за особенностей работы
сайта, они подписаны в коде. В связи с невозможностью отменить заказ на сайте, желательно не создавать много заказов и рекомендуется использовать
вновь созданный аккаун(создать можно с помощью "test_registration")
Также не забудьте установить пакеты из файла "requirements" это можно сделать командой "pip install -r requirements.txt"
Команда для запуска тестов с помощью pytest из папки 100sistem> python -m pytest -sv .\tests\test_critiсal_path.py (вместо последнего, можно указать нужный тест)

Проект автоматизации тестирования кртитического пути сайта "https://100sistem.ru/"
Модули в блоке "pages" разбиты на страницы, у некоторых указаны url в классах
Модуль "base_cl" в блоке "base" содержит методы, которые могут использоваться во всех блоках
Модули в блоке "tests" необходимы для запуска тестов и разбиты по функциональной логике:
"test_registration" - создает аккаун пользователя, данные об УЗ будут доступны в тестовом виде в пайчарме
"test_login" - проверяет авторизацию по логину и карте лояльности, проверяемые тестовые данные необходимо указать в нужные переменные в try
"test_critical_path" - запускает проверку основного пути пользователя с тестовыми данными(указываются в переменные в try)
